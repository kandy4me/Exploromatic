Introducting the explor-O-matic

The explorOmatic is an autonomous RC Truck based around a Beaglebone Black and Robot Operating System (ROS). It has no aim in particular, but serves to be a platform for experimenting. In this vein it should have bi-directional communication and a good interface, which is accomplished through Ubuntu over a SSH connection. The initial version of the greenOmatic will essentially be a remote-controlled RC truck with all on-board sensors constantly being updated and reported. Lets call this version 1.0.

The next version of the greenOmatic will implement SLAM to create a precise map of the perimeter of any arbitrary indoor unit (multiple rooms). This could be useful to real-estate businesses for verifying the layout of floor plans. However, let's not get ahead of ourselves, Version 1.0 is still in the works!

Hardware

4WD Remote-Control Truck

I got very lucky on a trip to Value village, and found this great RC truck. It's 4-wheel-drive, has a decent suspension system, and has a pretty hefty drive motor. The steering motor was of an old Design (5 wire analog control), so rather than figure out how this works I swapped it out for a modern Servo.
